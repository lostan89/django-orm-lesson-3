# Скрипты для взлома электронного журнала

Репозиторий электронного журнала находится здесь --> https://github.com/devmanorg/e-diary/tree/master

<br>
Всего предcтавлено 3 скрипта:

**fix_marks** - исправляет оценки указанного ученика;

**remove_chastisements** - удаляет замечания указанного ученика;

**create_commentation** - добавляет одну случайную похвалу для указанного ученика по предмету.

### Как использовать

1. Чтобы использовать скрипт **fix_marks**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента `child` укажите часть или полное ФИО требуемого ученика.
Если скрипт найдет несколько учеников с частично совпавшим ФИО или не найдет ни одного - скрипт выдаст соответствующую ошибку (проверка происходит в функции **find_schoolkid**).

По умолчанию все отрицательные оценки этого ученика изменятся на 5 ("оценка пять"). 

Оценку можно изменить, поменяв значение поля points `schoolkid.update(points=5)`.

<br>

2. Чтобы использовать скрипт **remove_chastisements**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента `child` укажите часть или полное ФИО требуемого ученика.
Если скрипт найдет несколько учеников с частично совпавшим ФИО или не найдет ни одного - скрипт выдаст соответствующую ошибку (проверка происходит в функции **find_schoolkid**).

После выполнения скрипта все замечания этого ученика будут удалены.

<br>

3. Чтобы использовать скрипт **create_commendation**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента `child` укажите часть или полное ФИО требуемого ученика.
Если скрипт найдет несколько учеников с частично совпавшим ФИО или не найдет ни одного - скрипт выдаст соответствующую ошибку (проверка происходит в функции **find_schoolkid**).

В качестве аргумента `lesson` укажите название предмета, по которому нужно добавить похвалу.
После исполнения скрипта будет добавлена одна случайная похвала для указанного ученика.
