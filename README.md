# Скрипты для взлома электронного журнала

Репозиторий электронного журнала находится здесь --> https://github.com/devmanorg/e-diary/tree/master

<br>
Всего предcтавлено 3 скрипта:

**fix_marks** - исправляет оценки указанного ученика;

**remove_chastisements** - удаляет замечания указанного ученика;

**create_commentation** - добавляет одну случайную похвалу для указанного ученика по предмету.

### Как использовать

1. Чтобы использовать скрипт **fix_marks**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента `schoolkid` укажите отфильтрованный список объектов модели Marks c оценками меньше 4 по требуемому ученику.

Например, 

```schoolkid = Marks.objects.filter(points__lt=4, schoolkid="Фролов Иван Григорьевич 6А")```.

По умолчанию все отрицательные оценки изменяются на 5 ("оценка пять"). 

Это можно изменить, поменяв значение `bad_point.points`.

<br>

2. Чтобы использовать скрипт **remove_chastisements**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента 'schoolkid' укажите отфильтрованный список объектов модели Chastisement по требуемому ученику.

Например, 

`schoolkid = Chastisement.objects.filter(schoolkid="Фролов Иван Григорьевич 6А")`

После выполнения скрипта все замечания указанного ученика будут удалены.

<br>

3. Чтобы использовать скрипт **create_commentation**, скопируйте одноименную функцию из файла скрипта в *Django Shell*.

В качестве аргумента `child` укажите часть или полное ФИО требуемого ученика.
Если скрипт найдет несколько учеников с частично совпавшим ФИО или не найдет ни одного - скрипт выдаст соответствующую ошибку.

В качестве аргумента `lesson` укажите по название предмета, по которому нужно добавить похвалу.
После исполнения скрипта будет добавлена одна случайна похвала.
